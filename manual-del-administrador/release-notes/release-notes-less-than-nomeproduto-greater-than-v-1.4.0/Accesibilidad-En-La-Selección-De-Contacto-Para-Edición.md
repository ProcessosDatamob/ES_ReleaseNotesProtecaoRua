# Accesibilidad en la Selección de Contacto para Edición

**ID: US-BTC-468**

Descripción: En la aplicación, la pantalla "Seleccione un contacto para editar" ahora está adaptada para ser fácilmente utilizada por personas con discapacidad visual. Cuando un lector de pantalla está activo, la navegación entre los elementos ocurre de manera organizada y fácil de seguir. Cada componente se describe de forma clara, con identificación precisa de sus tipos, lo que permite que los usuarios elijan el contacto que desean editar sin dificultades.
